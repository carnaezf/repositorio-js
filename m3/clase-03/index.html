Identificación del Mayor y Menor

En este ejercicio deberá ingresar tres números mediante prompt(), almacenarlos en un arreglo y ordenarlos utilizando el método de ordenamiento burbuja.

Objetivo del ejercicio

  • Solicitar tres números al usuario.
  • Guardar los valores dentro de un arreglo.
  • Ordenar el arreglo usando burbuja.
  • Identificar el número menor y el número mayor.
  • Detectar si los tres números son iguales.

Resultado

Abra la consola o recargue la página para ejecutar el programa.
