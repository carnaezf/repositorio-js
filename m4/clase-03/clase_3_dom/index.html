Eventos y Manipulación del DOM

Hoy trabajaremos cómo JavaScript puede seleccionar, modificar, crear y reaccionar a elementos del documento HTML.

1. ¿Qué es el DOM?

El DOM es la representación del documento HTML como una estructura de objetos. JavaScript puede acceder a esa estructura para leer, modificar, agregar o eliminar elementos.

Idea clave: si HTML define la estructura, JavaScript puede modificar esa estructura en tiempo real.

Árbol DOM simplificado
  • window
  • document
  • html
  • head / body
  • header, main, section, button...

2. Seleccionar elementos del DOM

Antes de modificar un elemento, primero debemos seleccionarlo desde JavaScript.

Métodos tradicionales
  • getElementById()
  • getElementsByClassName()
  • getElementsByName()
  • getElementsByTagName()

Son métodos clásicos, disponibles desde versiones antiguas de JavaScript.

Métodos modernos
  • querySelector()
  • querySelectorAll()

Funcionan parecido a los selectores CSS y suelen ser más cómodos para trabajar.

5. Crear y agregar elementos

Podemos crear nuevos elementos con createElement() y agregarlos al DOM con appendChild() o métodos similares.

6. Evento onBlur

El evento blur ocurre cuando el usuario sale de un campo de formulario. Es útil para validar datos después de escribir.

Escriba un nombre y luego haga clic fuera del campo.

7. Evento onKeyDown

El evento keydown se ejecuta cuando el usuario presiona una tecla.

Aquí aparecerá la tecla presionada.

8. Evento submit

El evento submit ocurre cuando se envía un formulario. Con JavaScript podemos validar antes de procesar la información.
